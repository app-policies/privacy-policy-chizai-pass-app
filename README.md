# 知財Pass プライバシーポリシー

## 1. アプリについて

知財Passは、知的財産管理技能検定2級・3級の学習を支援する独立した試験対策アプリです。

本アプリは、日本知的財産教育協会、知的財産管理技能検定、特許庁、INPIT、その他の試験実施団体が提供、監修、承認、公認するアプリではありません。

本アプリは、弁理士業務、出願代理、権利侵害判断、商標調査代行、契約書レビュー、個別法務相談を行うものではありません。

## 2. 収集する情報

本アプリは、アプリ内での学習履歴、回答履歴、正誤情報、苦手問題の登録状況、連続学習日数、文字サイズ設定などを端末内に保存します。これらは学習機能を提供するために利用します。

本アプリは、出願内容、発明内容、契約書本文、商標調査依頼、個別相談内容、勤務先情報、氏名、住所、電話番号を入力させたり、当社が取得したりする機能を提供しません。

## 3. 広告と課金

本アプリでは、広告表示のために Google Mobile Ads SDK を利用する場合があります。広告配信に関する情報の取り扱いは、Google のポリシーに従います。

本アプリでは、プレミアム機能の購入管理のために RevenueCat および Apple In-App Purchase を利用する場合があります。購入処理は Apple の仕組みを通じて行われ、当社がクレジットカード番号などの決済情報を取得することはありません。

## 4. 第三者提供

法令に基づく場合を除き、当社が利用者を直接特定する情報を第三者へ販売または提供することはありません。
